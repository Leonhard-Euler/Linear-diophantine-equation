# Linear-diophantine-equation
A Linear-diophantine-equation problem solved.

### Counting the number of positive solutions for a linear diophantine equation.
#### Example
```
x + y = 0

input: [1,1], 0
output: 1
```
```
x + y = 1

input: [1,1], 1
output: 2
```

```
x + y = 2

input: [1,1], 2
output: 3
```

```
x + y = 3

input: [1,1], 3
output: 4
```

```
x + y + z + k = 3

input: [1,1,1,1], 30
output: 5456
```
